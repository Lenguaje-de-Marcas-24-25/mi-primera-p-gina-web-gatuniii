    Lista de descripciones

    W3C: Organización internacional que desarrolla estándares web.

    WHATWG: Grupo que trabaja en la evolución de HTML y APIs.

    HTML: Lenguaje de marcado estándar para la creación de páginas web.

    XHTML: Versión más estricta de HTML basada en XML.

    DTD: Define la estructura de un documento HTML o XML.

    APIs: Conjunto de herramientas y protocolos para la comunicación entre aplicaciones.

    Drag and Drop: API que permite arrastrar y soltar elementos dentro de una página web.

    Geolocalización: API que permite obtener la ubicación geográfica del usuario.

    Almacenamiento local: API que permite almacenar datos en el navegador.

    App Cache: Método para almacenar archivos localmente y usarlos sin conexión.

    WebWorkers: Permiten ejecutar scripts en segundo plano sin afectar el rendimiento de la página.

    Editor WYSIWYG: Editor de texto que permite ver cómo se verá el contenido mientras se edita.

    Navegadores: Aplicaciones que permiten visualizar y navegar por la web.

    Mozilla Firefox: Navegador desarrollado por Mozilla.

    Google Chrome: Navegador desarrollado por Google.

    Safari: Navegador desarrollado por Apple.

    Opera: Navegador conocido por sus innovaciones en funciones de navegación.
